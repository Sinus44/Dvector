Создание вектора

  1. Нажимаем на модуль "Вектор" правой кнопкой мыши (ПКМ).

  2. В всплывающих окнах указываем координату X, затем координату Y.

Модули

  1. Вектор

    •  Создание вектора, инструкция по созданию приведена выше.

  2. Нормализация

    •  Превращение любого вектора в единичный (Длина = 1ед).

  3. Сложение

    •  Сложение двух любых векторов.

  4. Умножение

    •  Умножение двух векторов друг на друга.

  5. Отображение

    •  Отображение одного вектора или других модулей.

  6. Вычитание

    •  Вычитание одного вектора из другого. Первым подключается тот, из которого вычитаение будет произведено.

  7. Число

    •  Хранит в себе число.

  8. Установка вращения

    •  Установка вращения вектора. Угол указывается в радианах в модуле "Число". Затем модуль "число" подключается к этому модулю.

  9. Начальная точка

    •  Точка, откуда будет отходить начало вектора.

Настройки

  1. Выбор языка

    1. Стандартно стоит русский язык.

    2. Для смены языка используйте кнопку "Выбор языка".

    3. Доступные для ввода языки - английский и русский.

    4. Для подключения русской базы введите в всплывающее окно "ru".

    5. Для подключения английской базы введите в всплывающее окно "en".

  2. Выбор темы

    1. Стандартно стоит тёмная тема.

    2. Для смены темы используйте кнопку "Смена темы".

    3. Доступные для ввода языки - тёмная и светлая.

    4. Для подключения светлой темы введите в всплывающее окно "white".

    5. Для подключения тёмной темы введите в всплывающее окно "black".

  3. Очистка всех соединений

    1. Данная кнопка удаляет все связи модулей и очищает вашу координатную плоскоть.

    2. Для выполнения достаточно нажать на кнопку левой кнопкой мыши (ЛКМ).

Управление

Для перемещения в координатном пространстве используются клавиши W, A, S, D.
